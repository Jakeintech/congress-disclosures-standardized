# Utility Scripts

Helper scripts for data validation, schema checking, and debugging.

## Scripts

- `check_bill_data.py` - Validate Congress.gov bill data
- `check_dim_members.py` - Check member dimension data quality
- `check_fact_filings.py` - Validate filing fact table
- `check_house_dim_members.py` - House-specific member checks
- `check_schema_s3.py` - S3 schema validation

These are development/debugging utilities, not part of the production pipeline.
